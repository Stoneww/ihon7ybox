
# 创建Layui栅格布局

### 1.创建路由

```
修改 t00ls/urls.py 文件，添加一行路由:
    path('hello/', views.hello, name='hello'),
```

### 2.创建视图模板

```
修改 t00ls/views.py 文件，修改路由 /t00ls/hello/ 由 hello.html文件渲染。
def hello(request):
    return render(request, 't00ls/hello.html')
```
